# H5开发问题汇总

- 使用px2rem类似转换包之后,圆点变形

  结合transform和scale使用

```css
  .test{
    width:1px;
    height:1px;
    transform:scale(10)
  }
```

- 部分输入框文字不展示，但是有占位。造成的原因大概率是系统的护眼模式等类似的主题造成的

  全局增加字体颜色

- 全局设置标签颜色，不然手机主题可能导致颜色显示问题
