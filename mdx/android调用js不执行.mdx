# Vue生命周期中，Android不能正常调用JS代码

  嵌入App的H5 Vue项目需要获取Token,App提供了方法供H5调用,在Vue生命周期中，Android不能正常调用JS代码

```vue
  onMounted(()=>{
    // uboxClient.getToken 为App注入的方法
    window.uboxClient.getToken(function (token) {
      // 回调不能被正常执行
    })
  })
```
