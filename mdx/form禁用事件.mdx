# 禁用form默认事件

```js:禁用enter事件
	let forms = document.getElementsByTagName('form')
	let len = forms.length;
	for(let i = 0 ;i<len;i++){
		forms[i].addEventListener('keydown',function(event){
			if(event.key=='Enter'){
				event.preventDefault()
			}
		})
	}

```
