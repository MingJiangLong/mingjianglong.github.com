# Typescript类型学习

### 枚举类型提取
```ts
enum NAME {
  'xiaoming'='xiaoming',
  'xiaozhang'='xiaozhang',
}
type Name = keyof typeof Name// 'xiaoming'|'xiaozhang'
```
